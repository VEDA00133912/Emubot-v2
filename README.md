## 起動
1・[.env.sample](.env.sample)にbotのToken,APIの必要事項を入力<br>
2・.sampleの部分を消す。<br>
3・[config.json](lib/data/config.json)にIDとlogChannelIdを入力<br>
4・```node index.js``` で起動します
